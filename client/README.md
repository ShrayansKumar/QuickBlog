# QuickBlog 📝

A full-stack AI-powered blogging platform built with React, Node.js, Express, MongoDB, and Groq AI.

## Features

- 📰 Create, edit, and publish blog posts
- 🤖 AI-powered blog content generation using Groq
- 🔐 Admin authentication with JWT
- 🖼️ Image upload via Multer
- 💬 Comment system
- 🌐 REST API backend

## Tech Stack

**Frontend**
- React + Vite
- Tailwind CSS

**Backend**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Multer (image uploads)
- Groq AI (content generation)
